Generador de Contraseñas - Documentación
Descripción del Proyecto
Este es un generador de contraseñas seguro y personalizable desarrollado con HTML, CSS y JavaScript puro. La aplicación permite a los usuarios crear contraseñas robustas con diferentes combinaciones de caracteres y ajustar la longitud según sus necesidades.

Características
✅ Generación de contraseñas aleatorias y seguras

✅ Personalización de parámetros:

Longitud de contraseña (4-30 caracteres)

Inclusión de mayúsculas (A-Z)

Inclusión de minúsculas (a-z)

Inclusión de números (0-9)

Inclusión de símbolos (!@#$%^&*)

✅ Interfaz de usuario intuitiva y responsive

✅ Funcionalidad de copiar al portapapeles

✅ Diseño moderno y atractivo
